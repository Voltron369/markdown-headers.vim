4. CONFIGURATION                     *markdown-headers-configuration*

All configuration is done through global variables in your |vimrc|.

------------------------------------------------------------------------------
4.1 Font Style                       *g:md_h1_style* *g:md_h2_style* 
                                     *g:md_h3_style* *g:md_h4_style*
                                     *g:md_h5_style* *g:md_h6_style*

Set the font style for each header level (H1-H6).

Valid values:~
  'doublestruck' - Use double-struck Unicode characters (ð”¸ð”¹â„‚)
  'fullwidth'    - Use fullwidth Unicode characters (ï¼¡ï¼¢ï¼£)
  'none'         - Use normal characters (ABC)

Default:~
  All levels default to 'doublestruck'

Example:~
>
    let g:md_h1_style = 'fullwidth'
    let g:md_h2_style = 'doublestruck'
    let g:md_h3_style = 'none
<

------------------------------------------------------------------------------
4.2 All Caps                         *g:md_h1_caps* *g:md_h2_caps*
                                     *g:md_h3_caps* *g:md_h4_caps*
                                     *g:md_h5_caps* *g:md_h6_caps*

Enable or disable all-caps display for each header level.

Valid values:~
  1 - Enable all caps (convert lowercase to uppercase)
  0 - Disable (preserve original case)

Default:~
  H1: 1 (all caps enabled)
  H2-H6: 0 (all caps disabled)

Example:~
>
    let g:md_h1_caps = 1
    let g:md_h2_caps = 1
    let g:md_h3_caps = 0
<

------------------------------------------------------------------------------
4.3 Complete Configuration Example

>
    " H1: Fullwidth, all caps
    let g:md_h1_style = 'fullwidth'
    let g:md_h1_caps = 1

    " H2: Double-struck, all caps
    let g:md_h2_style = 'doublestruck'
    let g:md_h2_caps = 1

    " H3-H6: Double-struck, normal case
    let g:md_h3_style = 'doublestruck'
    let g:md_h3_caps = 0
    " (h4, h5, h6 will use defaults if not specified)
<

5. COMMANDS                             *markdown-headers-commands*

                                            *:MarkdownHeadersToggle*
:MarkdownHeadersToggle      Toggle the display of Unicode headers on/off.
                            This command toggles between 'conceallevel' 2
                            and 0, allowing you to quickly see the original
                            markdown syntax or the Unicode display.

Add this to your vimrc:~

command! MarkdownHeadersToggle call s:ToggleHeaders()

function! s:ToggleHeaders()
  if &conceallevel == 0
    setlocal conceallevel=2
  else
    setlocal conceallevel=0
  endif
endfunction
<

You can map this to a key for quick access:~
>
    nnoremap <leader>mh :MarkdownHeadersToggle<CR>
<

7. TROUBLESHOOTING                 *markdown-headers-troubleshooting*

Problem: Unicode characters don't display~
Solution: Ensure your terminal and font support Unicode. Try switching to
          a modern monospace font like JetBrains Mono or Fira Code.

Problem: Characters display but look wrong~
Solution: Your font may not include the specific Unicode blocks. Try a
          different font or switch between 'doublestruck' and 'fullwidth'.

Problem: Headers aren't transforming~
Solution: Check that concealing is enabled:
>
          :set conceallevel?
<
          Should return 2. Also check:
>
          :set concealcursor?
<
          If you want concealing even when cursor is on the line, try:
>
          :set concealcursor=nvc
<

Problem: Want to see original markdown temporarily~
Solution: Use |:MarkdownHeadersToggle| or manually:
>
          :set conceallevel=0
<

Problem: Changes in vimrc not taking effect~
Solution: Ensure variables are set before the plugin loads, or reload:
>
          :e
<

Problem: Interferes with other markdown plugins~
Solution: This plugin uses after/syntax/ which loads last. If conflicts
          occur, try adjusting the load order or disabling competing
          syntax modifications.
